## 一、机制说明

### tagCharge（同类集群互充）
- 某物品拥有 `tagCharge: {tag: X, ms: N}` 时，意味着：
  - **当该物品触发**：为场上所有其他同 tag X 的物品充能 N ms
  - **当其他同 tag X 物品触发**：该物品同样被充能 N ms（对称互充）
- 集群件数越多、CD 越短，互充效果雪球越明显（"左脚踩右脚"）

### 其他充能类型
| 属性名 | 方向 | 效果 |
|--------|------|------|
| `waterCharge` | 接收 | 任意水系物品触发时，自身充能 N ms |
| `chargeWaterItems` | 给出 | 自身触发时，所有水系物品充能 N ms |
| `globalCharge` | 给出 | 自身触发时，所有其他物品充能 N ms |
| `companionCharge` | 给出 | 自身触发时，所有伙伴物品充能 N ms |
| `adjacentCharge` | 给出 | 自身触发时，左右相邻物品各充能 N ms |
| `chargeFromAdjacentWeapon` | 接收 | 左右相邻武器每次触发，自身充能 N ms |
| `chargeReadiest` | 给出 | 自身触发时，为进度最高的其他物品充能 N ms |

| 阶数 | 冷却 | 灼烧 | 特殊伤害 |

## 四、强力组合推荐

### ★★★ S 级

---

#### 1. 「火焰机器」— 打火机 + 燃烧响炮 + 风暴瓶

**类型**：工具三件套，灼烧雪球

**运作逻辑**：
1. 三件工具 tagCharge 互充 0.6s，整体加速约 1.45×
2. 打火机触发 → 施灼烧 + 余烬反噬（灼烧层×0.5 直接伤害）
3. 燃烧响炮触发 → 敌方已有灼烧 → 本次灼烧 ×1.5
4. 风暴瓶触发 → 全场所有物品充能 1s，整体加速

**雪球路径**：灼烧层↑ → 打火机余烬伤害↑ → 更快叠灼烧 → 风暴瓶加速循环更快 → 灼烧层↑↑

**注意**：开局前几秒灼烧层少，中后期指数级爆发。适合中长时间战斗。

---

#### 2. 「左轮无限」— 左轮手枪 + 连发步枪（相邻）

**类型**：枪械双核，自成长加速机制

**运作逻辑**：
1. 左轮每触发 6 次 CD 永久减少（Diamond 阶：3s → 1s 仅需 1 轮 6 发）
2. 连发步枪从相邻武器触发中充能（Diamond 阶：每次 2s）
3. 左轮加速后频繁触发 → 连发步枪被不断充能 → 步枪也趋向无限触发

**加速示意**（Diamond 阶，相邻放置，60s 后）：
- 左轮：6 发减 2s，共约 6 轮 → CD 降至最低 1s
- 步枪：每次左轮触发充能 2s，CD 6s → 实际约 2s 触发一次

**注意**：这是枪械唯一强力组合；两枪无 tagCharge，必须靠相邻位置保证步枪充能。

---

#### 3. 「爆炸回路」— 弯刀 + 手雷（相邻放置）

**类型**：武器双核，链式充能回路

**运作逻辑**（弯刀与手雷相邻）：
1. 弯刀触发 → 随机链击 → 若只有手雷则必定触发手雷
2. 手雷触发 → 相邻充能 2s 给弯刀 + 武器 tagCharge 0.6s 给弯刀
3. 弯刀获得 2.6s 充能 → 更快再次触发 → 再次链击手雷

**效果**：弯刀实际 CD 从 5s 压缩至约 2.4s，手雷实际 CD 从 7s 压缩至约 4.4s（受武器 tagCharge 0.6s 加速）。

**加入连发步枪（相邻）**：弯刀/手雷每次触发都给步枪充能，步枪获得极高频率触发。

### ★★ A 级

#### 4. 「不沉之躯」— 珊瑚护甲 + 潜水头盔 + 珍珠

**类型**：防具+水系双集群叠盾

- 珊瑚护甲：每次触发护盾递增，战斗越长护盾越高

**加分项**：加入珊瑚（食物），HP 满时超量治疗转盾，进一步叠护盾上限。

#### 5. 「海怪军团」— 鹦鹉 + 幽渊鮟鱇 + 河豚 + 食人鱼（4 件伙伴）

**类型**：伙伴集群核心输出

- 每件伙伴触发均为其余 3 件充能 0.5s（鹦鹉触发额外 +1s 给所有伙伴）
- 幽渊鮟鱇：为最高进度物品充能 3s，智能加速

**配合鱼饵（食物）**：鱼饵持续叠全局暴击率，加强所有伙伴伤害输出。

---

#### 6. 「满血成盾」— 珊瑚 + 海草 + 鱼饵（食物三件套）

**搭配建议**：加入潜水头盔（水系充能加速海草），组成食物+水系+防具的复合体系。
