# 中级档技能（2个）

> 更新时间：2026-03-05

`level: "intermediate"`，`trigger: "on_item_trigger"`，道具触发时额外执行。

---

## 血刃回响 `blade_echo` ⚔️

**效果**：每次造成伤害的道具触发时，额外造成 X 点伤害

| 品质 | 额外伤害 |
|---|---|
| 铜 | 8 |
| 银 | 18 |
| 金 | 32 |
| 钻石 | 50 |

**触发条件**：道具本次触发结算的最终 `damage > 0`（含 bonus）

---

## 毒雾蔓延 `poison_spread` 💀

**效果**：每次施加剧毒的道具触发时，额外施加 X 层剧毒

| 品质 | 额外剧毒层数 |
|---|---|
| 铜 | 1 |
| 银 | 2 |
| 金 | 4 |
| 钻石 | 6 |

**触发条件**：道具本次触发结算的最终 `poison > 0`（含 bonus）
