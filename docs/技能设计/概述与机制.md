# 技能系统 — 概述与机制

> 更新时间：2026-03-05

---

## 一、技能分类

技能数据存储于 `config/skills.json`，共 10 个技能，分两大类：

| 类型 | category | 数量 | 获取方式 |
|---|---|---|---|
| 章节技能 | `chapter` | 7 | 每章 BOSS 战后 2 选 1 |
| 身份技能 | `identity` | 3 | 选择身份时绑定，固定拥有 |

---

## 二、章节技能

### 获取机制

- 每章第 4 场战斗（`BATTLES_PER_CHAPTER = 4`）为 BOSS 战
- BOSS 战结束后弹出技能选择界面（`SKILL_SELECT`），展示 2 个随机候选
- 候选从未拥有的 `category: "chapter"` 技能中随机抽取
- 品质（tier）由当前章节决定，见下表

### 章节与技能品质

```
CHAPTER_SKILL_TIERS = ["Bronze", "Silver", "Gold", "Diamond"]
```

| 章节 | 名称 | 技能品质 |
|---|---|---|
| 第 1 章 | 启航 | 铜 |
| 第 2 章 | 风暴 | 银 |
| 第 3 章 | 深渊 | 金 |
| 第 4 章 | 宝藏岛 | 钻石 |
| 第 5 章 | 传奇海域 | 钻石 |

### 章节技能列表

| 技能 | id | 等级档 | 触发时机 |
|---|---|---|---|
| 潮涌护甲 | `tidal_armor` | 基础 | 战斗开始 |
| 幽影剧毒 | `shadow_poison` | 基础 | 战斗开始 |
| 炼狱先手 | `inferno_start` | 基础 | 战斗开始 |
| 先发制人 | `first_strike` | 基础 | 战斗开始 |
| 血刃回响 | `blade_echo` | 中级 | 道具触发时 |
| 毒雾蔓延 | `poison_spread` | 中级 | 道具触发时 |
| 不屈之魂 | `undying` | 高级 | 致命伤害拦截 |

详见各档技能文档。

## 三、身份技能

身份技能随身份选择自动获得，固定为 **铜** 品质，贯穿全局生效（不受章节影响）。

| 身份 | 技能 id | 技能名 | 触发类型 |
|---|---|---|---|
| 军火商人 | `gunner_passive` | 先制炮击 | 战斗开始，按枪械数量造成伤害 |
| 深海渔翁 | `tamer_passive` | 深海护甲 | 伙伴×3：每次触发获得护盾 |
| 流浪炼金士 | `alchemist_passive` | 毒焰协同 | 工具×3：每次触发附加灼烧+剧毒 |

详见「身份技能.md」。

## 四、章节事件权重

每章除技能外，还会对特定事件的出现概率进行加权（`CHAPTER_EVENT_BONUS_WEIGHT = 3`）或屏蔽：

| 章节 | 战后加权 | 战前加权 | 战后屏蔽 | 战前屏蔽 |
|---|---|---|---|---|
| 启航 | 残骸搜寻 | 以物换物 | 宝藏商店 / 风暴洗牌 / 破釜沉舟 | 深锚强化 |
| 风暴 | 残骸搜寻 / 蒙面交易 | 以物换物 / 深锚强化 | 宝藏商店 / 风暴洗牌 | — |
| 深渊 | 宝藏商店 / 蒙面交易 / 赌徒骰子 | 指定升级 / 以物换物 / 深锚强化 | 风暴洗牌 | — |
| 宝藏岛 | 风暴洗牌 / 破釜沉舟 / 宝藏商店 | 指定升级 / 炼金 / 深锚强化 | — | — |
| 传奇海域 | 破釜沉舟 / 蒙面交易 / 酒馆赌注 | 炼金 / 指定升级 / 深锚强化 / 以物换物 | 风暴洗牌 / 残骸搜寻 | — |

加权事件的抽取权重乘以 3，被屏蔽的事件权重为 0（不会出现）。

## 五、数据结构

### 技能字段（skills.json）

```json
{
  "id":        "tidal_armor",
  "icon":      "🌊",
  "name_cn":   "潮涌护甲",
  "desc_cn":   "战斗开始时获得 {value} 点护盾",
  "category":  "chapter",
  "level":     "basic",
  "trigger":   "battle_start",
  "effect":    "shield",
  "tiers": {
    "Bronze":  { "value": 25 },
    "Silver":  { "value": 50 },
    "Gold":    { "value": 85 },
    "Diamond": { "value": 130 }
  }
}
```

| 字段 | 说明 |
|---|---|
| `category` | `chapter`（章节技能）/ `identity`（身份技能） |
| `level` | 章节技能档次：`basic` / `intermediate` / `advanced` |
| `trigger` | 触发时机：`battle_start` / `on_item_trigger` / `on_lethal` |
| `condition` | 触发条件（可选）：`damage` / `poison` 等 |
| `effect` | 效果类型：`shield` / `poison` / `burn` / `damage` / `survive` |
| `tiers` | 各品质对应数值，`desc_cn` 中用 `{value}` 占位 |
