# 身份技能（3个）

> 更新时间：2026-03-05

`category: "identity"`，随身份选择自动绑定，固定铜品质，全局常驻。

---

## 先制炮击 `gunner_passive` 💣

**所属身份**：军火商人

**效果**：战斗开始时，每件枪械道具造成 {value} 点伤害

| 品质 | 每件枪械伤害 |
|---|---|
| 铜 | 15 |
| 银 | 25 |
| 金 | 40 |
| 钻石 | 60 |

**实现**：`type: "preemptive_strike"`，`tags: ["枪械"]`
触发时遍历 playerItems，统计 `tags.includes("枪械")` 的物品数量，每件造成 value 点伤害。

---

## 深海护甲 `tamer_passive` 🛡️

**所属身份**：深海渔翁

**效果**：阵容中伙伴类道具 ≥ 3 件时，每次任意伙伴道具触发后获得 {value} 点护盾

| 品质 | 每次触发护盾 |
|---|---|
| 铜 | 8 |
| 银 | 15 |
| 金 | 25 |
| 钻石 | 40 |

**实现**：`type: "tag_shield_on_trigger"`，`tags: ["伙伴"]`，`threshold: 3`
触发条件：playerItems 中 `tags.includes("伙伴")` 的数量 ≥ threshold。

---

## 毒焰协同 `alchemist_passive` ⚗️

**所属身份**：流浪炼金士

**效果**：阵容中工具类道具 ≥ 3 件时，每次任意工具道具触发后附加 {value} 层灼烧和剧毒

| 品质 | 每次触发灼烧 / 剧毒 |
|---|---|
| 铜 | 各 1 层 |
| 银 | 各 2 层 |
| 金 | 各 3 层 |
| 钻石 | 各 5 层 |

**实现**：`type: "tag_burn_poison_on_trigger"`，`tags: ["工具"]`，`threshold: 3`
触发条件：playerItems 中 `tags.includes("工具")` 的数量 ≥ threshold。
