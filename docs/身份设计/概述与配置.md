# 身份系统 — 概述与配置

> 更新时间：2026-03-05

---

## 一、系统概述

身份在游戏开始时选择（`IDENTITY` 阶段），决定：

1. **初始物品**：1 件固定道具 + 2 件同标签随机道具
2. **被动技能**：绑定一个 `category: "identity"` 的技能，铜品质，贯穿全局

身份数据存储于 `config/identities.json`。

---

## 二、字段规范

```json
{
  "id":        "gunner",
  "icon":      "💣",
  "portrait":  "identities/gunner.jpg",
  "name":      "军火商人",
  "flavor":    "「好枪配好手」",
  "startItems": [6],
  "startTag":  "枪械",
  "itemNames": ["左轮手枪", "+ 2件随机枪械类"],
  "skillId":   "gunner_passive",
  "skillTier": "Bronze"
}
```

| 字段 | 说明 |
|---|---|
| `startItems` | 固定初始物品 ID 数组（取第一个） |
| `startTag` | 随机补充物品的类型标签，从该类型物品池中随机取 2 件 |
| `itemNames` | 仅供 UI 展示，不参与逻辑 |
| `skillId` | 绑定的身份技能 ID（来自 skills.json） |
| `skillTier` | 技能初始品质，目前固定为 `Bronze` |

---

## 三、当前三个身份

### 军火商人（gunner）💣

- **初始物品**：左轮手枪 + 2 件随机枪械
- **被动技能**：先制炮击 — 战斗开始时，每件枪械造成 15 点伤害
- **定位**：纯攻击，开局爆发；枪械数量越多，先手优势越大

---

### 深海渔翁（tamer）⚓

- **初始物品**：食人鱼 + 2 件随机伙伴
- **被动技能**：深海护甲 — 伙伴×3 时，每次触发获得 8 点护盾
- **定位**：持续防御，伙伴数量堆到 3 件后护盾滚雪球效果显著

---

### 流浪炼金士（alchemist）⚗️

- **初始物品**：打火机 + 2 件随机工具
- **被动技能**：毒焰协同 — 工具×3 时，每次触发附加 1 层灼烧 + 1 层剧毒
- **定位**：双 DoT 叠层，工具类触发频率高时毒焰累积效果强

---

## 四、初始物品分配逻辑

```js
// App.vue — onIdentityChoice
const fixedItem = startItems[0]  // 固定道具（取第一个）
const tagPool   = ITEM_POOL.filter(i => i.tags?.includes(startTag) && i.id !== fixedItem.id)
shuffle(tagPool)
const itemsToGive = [fixedItem, ...tagPool.slice(0, 2)]  // 共 3 件
```

所有初始物品均为铜品质，投放到背包/阵容中。

---

## 五、身份技能展示

身份技能在 ARRANGE 界面状态栏中展示，点击可查看完整描述（tooltip）。
描述文本由 `desc_cn` 模板填充当前 tier 的 `value` 生成：

```js
base.desc_cn.replace('{value}', base.tiers[skill.tier].value)
```
