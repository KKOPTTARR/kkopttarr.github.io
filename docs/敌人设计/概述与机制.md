# 敌人系统 — 概述与机制

> 更新时间：2026-03-04

---

## 一、系统概述

敌人数据存储于 `config/enemies.json`，共 20 个敌人模板。
每场战斗随机从模板池抽取 3 个敌人，分别赋予 **normal / elite / radiant** 三种难度供玩家选择。
敌人 HP 与技能伤害随战斗轮次自动缩放，难度越高奖励越好。

---

## 二、字段规范

### 敌人模板结构

```json
{
  "id":       "parrot",
  "name":     "鹦鹉船员",
  "portrait": "enemies/parrot.png",
  "hp": {
    "normal":  140,
    "elite":   187,
    "radiant": 218
  },
  "normalAbilities":  [...],
  "eliteAbilities":   [...],
  "radiantAbilities": [...],
  "dropBias": ["工具"]
}
```

### 技能结构

```json
{
  "id":          "parrot-n1",
  "name":        "利喙啄击",
  "cooldown":    3000,
  "damage":      6,
  "heal":        0,
  "shield":      0,
  "burn":        0,
  "poison":      0,
  "description": "每3秒造成4伤害"
}
```

| 字段 | 类型 | 说明 |
|---|---|---|
| `cooldown` | ms | 技能冷却（毫秒） |
| `damage` | int | 直接伤害 |
| `heal` | int | 回复血量（当前均为 0） |
| `shield` | int | 护盾值（当前均为 0） |
| `burn` | int | 附加灼烧层数（每层每秒 1 伤害） |
| `poison` | int | 附加剧毒层数（每层每秒 1 伤害，可叠加） |

## 三、难度机制

### 技能叠加规则

| 难度 | 技能组合 |
|---|---|
| normal | normalAbilities |
| elite | normalAbilities + eliteAbilities |
| radiant | normalAbilities + eliteAbilities + radiantAbilities |

难度越高，敌人同时拥有更多技能并行触发，输出压力线性叠加。

### HP 与伤害缩放公式

> 缩放在 `buildEnemy(template, difficulty, battleCount)` 中执行。

```js
const hpScale  = Math.pow(1.3, battleCount)   // HP 每轮 ×1.3
const dmgScale = Math.pow(1.2, battleCount)   // 伤害 每轮 ×1.2

actualHp     = hp[difficulty]   × hpScale  × 0.7
actualDamage = damage           × dmgScale × 0.7
actualBurn   = burn             × 0.7  (取整，最低 0)
actualPoison = poison           × 0.7  (取整，最低 0)
```

**基础系数 0.7**（`ENEMY_BASE_MULT`）整体压低数值，保证玩家早期不过于压迫。

### 各轮次倍率参考

| 轮次 | battleCount | HP 倍率 | 伤害倍率 |
|---|---|---|---|
| 1 | 0 | ×0.70 | ×0.70 |
| 2 | 1 | ×0.91 | ×0.84 |
| 3 | 2 | ×1.18 | ×1.01 |
| 4 | 3 | ×1.54 | ×1.21 |
| 5 | 4 | ×2.00 | ×1.45 |
| 6 | 5 | ×2.60 | ×1.74 |
| 7 | 6 | ×3.38 | ×2.09 |

## 四、奖励规则

奖励数据来自 `config/rewards.json`：

| 难度 | 掉落数量 | Bronze | Silver | Gold |
|---|---|---|---|---|
| normal | 1 | 100% | — | — |
| elite | 2 | 75% | 25% | — |
| radiant | 2 | — | 75% | 25% |

`dropBias` 字段限定掉落偏向的物品标签（如 `["武器", "工具"]`），在 `dropBias` 范围内优先抽取。
