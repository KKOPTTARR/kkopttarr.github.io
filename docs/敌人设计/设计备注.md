## 一、数值一致性问题

以下问题来自对 `config/enemies.json` 的梳理，供后续迭代参考。

| 问题 | 涉及敌人 | 说明 |
|---|---|---|
| `burn/poison ×0.7` 取整后为 0 | parrot 精英 burn:1 → 0 | 低层数状态效果被 BASE_MULT 归零，失去设计意图 |
| 立绘路径错误 | bat-demon | portrait 指向 `lava-golem.png`，非 bat 专属图 |
| 立绘路径错误 | demon-boss | portrait 指向 `mimic.png` |
| 精英技能完全重复 | gargoyle / bat-demon | 技能名、数值均完全一致（暗影屏障），缺乏差异化 |
| description 与实际数值不符 | rat / parrot / crab 等多处 | description 字段为旧版文本，未随 config 数值同步更新 |
| 无吸血机制 | vampire | heal 字段全为 0，与「吸血鬼」主题不符 |
